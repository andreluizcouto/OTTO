**src/lib/auth.ts** — gerenciamento de token:
```typescript
const TOKEN_KEY = 'fincoach_access_token';
export const getToken = (): string | null => localStorage.getItem(TOKEN_KEY);
export const setToken = (token: string): void => localStorage.setItem(TOKEN_KEY, token);
export const clearToken = (): void => localStorage.removeItem(TOKEN_KEY);
export const isAuthenticated = (): boolean => !!getToken();
```

**src/lib/api.ts** — cliente fetch centralizado:
- Constante BASE_URL = 'http://localhost:8001'
- Função `apiFetch(path, options?)` que:
  1. Lê token com getToken()
  2. Faz fetch para `${BASE_URL}${path}` com headers Content-Type: application/json + Authorization: Bearer <token> se token presente
  3. Se response não ok: tenta parsear JSON com campo `detail`, lança Error(detail) ou Error(`HTTP ${status}`)
  4. Retorna response.json()
- Exports nomeados: `apiFetch` e helpers `apiGet(path)`, `apiPost(path, body)`, `apiPut(path, body)`

Não usar axios — fetch nativo apenas. Não criar classe — funções simples.
